# Usage

## To Install Dependencies

```
npm install
```

# To run the application


## Run in dev mode
```
npm run dev
```

# Database Seeder

> If you want to seed the data independently into the database ,then use


```
# Add all data in Database
node seeder -add

# Delete all data into Databasw
node seeder -delete
```
